# 平台品牌配置设计

平台名称由 PostgreSQL 配置统一驱动登录页品牌名、侧栏品牌名、网页标题和登录页页脚。增加无需登录的只读公共配置接口，只返回名称、Logo URL、登录背景 URL 和语言，不返回任何敏感配置。

管理员通过独立上传接口上传 PNG/JPEG。Logo 限制 2 MB、16–2000 像素；登录背景限制 8 MB、最小 1920×1080、最大 8000×8000。图片保存到 `/data/simpleHPC/uploads/platform`，数据库只保存 URL。

全局 `platform-ui.js` 在所有页面加载公共配置并应用。登录背景使用图片加半透明遮罩，保证表单可读。

语言当前固定 `zh-CN`。设置页显示简体中文，以及禁用的“English（待开发）”“繁体中文（待开发）”。
